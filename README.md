## Overview

HybridRFComm is a Final Year Research Project that proposes a hybrid RF-based off-grid communication system capable of providing reliable communication in environments where cellular connectivity is unavailable.

The proposed system integrates RF Mesh Networking with Opportunistic Cloud Synchronization using AWS IoT Core to improve traveller safety and emergency response.

---

## Features

- RF Mesh Communication
- Multi-hop Routing
- Flutter Mobile Application
- USB-OTG Communication
- Self-powered Communication Nodes
- Real-time SOS Alerts
- Offline Local Storage
- Opportunistic Cloud Synchronization
- AWS IoT Core Integration
- MQTT Messaging
- Grafana Monitoring Dashboard
